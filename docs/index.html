DrData is a Shiny-based interactive platform for end-to-end machine learning workflows — designed for students and practitioners in data science and AI.

Install DrData View on GitHub install.packages("DrData")
Platform

Everything in one workflow

Ten integrated modules that take you from raw data to a deployed model — point, click, explore.

Import→ Preprocess→ EDA→ Model→ AutoML→ Compare→ Explain→ Deploy

Data Import

Load CSV, Excel, RDS, or TXT files. Built-in example datasets: iris, mtcars, Titanic.

Preprocessing

Imputation, scaling (z-score, min-max, log), one-hot encoding, outlier removal, near-zero variance.

Exploratory Data Analysis

8 interactive plot types, normality tests (Shapiro-Wilk, KS, AD, JB), and an auto-EDA report.

Regression

8 algorithms including Ridge, Lasso, Random Forest, GBM, SVM, Neural Net. Interaction terms supported.

Classification

8 classifiers with ROC curve, confusion matrix, precision, recall, F1. Binary and multiclass.

Clustering

K-Means, Hierarchical (ward.D2), DBSCAN. Silhouette score, elbow plot, Gower distance for mixed data.

AutoML

Automatic algorithm selection and comparison. Regression and classification. Prevents data leakage.

Model Comparison

Side-by-side metrics dashboard and performance heatmap across all AutoML runs.

Explainability

Native feature importance for RF, GBM, Tree, LM. Permutation importance for any model type.

Deployment

Save trained models as .rds, reload them, and predict on new CSV data. Secure file handling.

Installation

Ready in two lines

All required packages are checked automatically on first launch.

# From CRAN
install.packages("DrData")

# Development version from GitHub
remotes::install_github("mohsenmehdinia/DrData")

# Launch
library(DrData)
run_app()
Reference

Exported functions

DrData also exposes programmatic helpers for use in your own scripts.

run_app()

Launch the full DrData Shiny application in your browser or RStudio viewer.

ml_split(data, train_ratio, seed)

Reproducible train/test split. Returns a named list with train and test data frames.

ml_prepare_data(data, target, features)

Validate and clean a data frame for modelling. Drops incomplete rows automatically.

build_model_formula(target, features, ...)

Build a formula object with optional two-way interaction terms.

ml_metrics_regression(y_true, y_pred)

Compute RMSE, MAE, and R² for regression predictions.
